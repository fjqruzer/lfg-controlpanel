# LFG Dashboard

Sports event and venue management dashboard.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 16 + React 19
- Tailwind CSS + Material Tailwind
- ApexCharts

## Structure

```
src/
├── app/              # Pages (App Router)
├── components/       # UI components
├── services/         # API layer (currently mock)
├── data/             # Mock data
└── context/          # State management
```

## Features

- Event management
- Venue management
- User management
- Dashboard analytics
- Authentication
